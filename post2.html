Analysis Of Our Fortnight Budget

published: 2021-07-26

Background

An analysis of my fortnightly budget to get a clear picture of income and expenses

Objectives:

  • learn python libraries
  • learn data collection
  • learn data wrangling
  • learn data analysis
  • learn to interpret results

Link to project on Github

How it went

The first step was to clearly define the questions and goals for this project. I wanted to understand and improve how we spent and managed our money. This would ensure that we utilized all available finances efficiently to meet our overall goals. I wanted to represent my data in graphs. I decided to use matplotlib after some research.

I was working with array data sets, therefore I looked into different websites and stumbled on NumPy, which is a python library used for working with arrays. I used pip to install and import it by adding the import keyword.

Resources used to learn:

  • matplot lib documentation
  • youtube videos

Things learnt:

  • (matplotlib) pyplot interfaces
  • (matplotlib) Axis scales and ticks
  • (matplotlib) labelLing plots
  • (NumPy) array creation using the arange()function
  • (NumPy) array slicing and indexing
  • (NumPy) array iterating

Next was plotting the graph. This was achieved by setting the x and y labels. The graphs were so close together and didn't appear as neat as I wanted them to be, further research led to grid spec which is a function of maplotlib that allows us to specify the geometry of the grid where a subplot will be placed. To make the graphs easier to read and pretty, I added colour to the lines, markers and labels.

FINDING CORRELATION

I also decided to explore if there was a correlation between the data sets. after a bit of research, I came across a python library called scipy. Scipy provides more utility functions for optimization, stats and signal processing I used it to find Pearson correlation coefficient, to represent linear relationships between two datasets. I used a scatter plot to display the relationship between our groceries and takeout.

Resources used to learn:

  • scipy.org

Things learnt:

  • calculating correlation between variables in python
  • parametric correlation analysis

lastly to add a bit of spice I represented all the data sets in a pie chart.
