How it's going..

DATA ACQUISITION

I wanted to get country data relating to visas status so I found a website with this data. The website had undocumented API so I contacted the website creator and they were happy to give me end points I needed to hit to get the data .

API's

The first step was to understand how to hit API's and extract data then transform the data .I used python request module. the request returned was in JSON and had some dirty data. I then had to clean the data and transform the JSON file to pandas dataframe so as to insert into PSQL table

Went through official pandas docs but stack overflow seemed to be easier to get the information I needed to get started. I ran pip install sqlalchemy!

PANDAS AND SQLALCHEMY

Next was to to understand how to add data to database from an API. I ended up using pandas to insert data to postgress. Mainly used YouTube to watch some tutorials to learn the basics.
